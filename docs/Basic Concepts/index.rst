Basic Concepts
==============

How OpenZFS is put together and what its features actually do. Start with
:doc:`VDEVs </Basic Concepts/Pool Structure/VDEVs>` if you are new to ZFS —
pool layout is the one decision that is hardest to change later.

These pages explain concepts and trade-offs; the
`man pages <../man/index.html>`__ remain the reference for exact syntax and
options.
